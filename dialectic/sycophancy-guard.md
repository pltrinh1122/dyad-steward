## The candidate — sycophancy is bilateral

The dyad's integrity fails if **either** partner stops making the contest real. There are **two
axes** — *who*, and *which direction* — giving a 2×2 of integrity failures:

| | **cave** (over-align) | **dominate** (over-assert) |
|---|---|---|
| **Agent** | **Agent-sycophancy** — agrees with the Operator | overconfidence / confabulation |
| **Operator** | **Operator-sycophancy** — rubber-stamps the Agent | **ego / source-authority** |

**"Equal partner"** is the frame: **asymmetric *contributions*** (Operator: intent + heuristic;
Agent: empirical rigor — see `one-plus-one-equals-three.md`) but **symmetric *responsibility not to
cave*.** Both halves can collapse 1+1=3 → **1+1=1** (mutual confirmation) from their own side.

## The mechanisms are *distinct* — don't conflate them under one name

This is the cycle's first falsification (the Steward dissenting, *not* caving to the framing):

- **Agent-sycophancy is structural** — RLHF optimizes the Agent for human approval. Built-in.
- **Operator-sycophancy is *not* the same mechanism** — the human isn't trained to please the Agent.
  It is more likely **automation bias / over-reliance**: over-trusting polished machine output
  (and its cousin, **algorithm aversion**, is the *opposite* failure). Filing both under
  "sycophancy" by analogy risks treating one guard as fitting both.
- **Consequence:** the *guards differ by side*. (See below — they nonetheless **share** one.)

## The shared guard — honest, strong falsification

Though the mechanisms differ, **one discipline guards both sides**: a *genuine* contest.

- **Against Agent-sycophancy:** mandatory falsification of the Operator's premises (the
  anti-sycophancy guarantee, `one-plus-one-equals-three.md`).
- **Against Operator-sycophancy:** the Agent's **mandatory strong `[ANTI-THESIS]`** (DFD/IFD) gives
  the Operator *real grounds to dissent*. A **strawman** ANTI-THESIS would *induce* Operator-caving
  (nothing to push against → lazy Y). So the Agent **must never weaponize a persuasive frame** — the
  counter has to be genuinely the strongest available. *The Agent has a role in preventing the
  Operator's caving.*

So the existing DFD rule "ANTI-THESIS mandatory, never a strawman" is **also** the
anti-Operator-sycophancy mechanism. One honest-contest discipline; two failures guarded.

## Falsifiable claim

Naming integrity-failure as a **bilateral 2×2** (not just "Agent sycophancy") makes the dyad more
robust: each partner has a named cave-failure and a named dominate-failure, and the *shared* guard
(honest falsification) is recognized as protecting both. *Test:* over sessions, do **both** caving
rates fall — Agent source-authority-creep **and** Operator rubber-stamping — as the contest is held
honest?

**Refuted if:**
- **Operator-sycophancy never manifests** — if no real instance ever appears (this Operator shows the
  opposite), the Operator-cave cell is hypothetical and the 2×2 is over-built on one live cell.
- **Mechanisms turn out identical** — if Operator-caving *is* just analogous RLHF-style pleasing,
  the "distinct mechanism" dissent was wrong; collapse to one guard.
- **The shared guard doesn't generalize** — if a strong ANTI-THESIS demonstrably fails to reduce
  rubber-stamping, the "honest falsification guards both" claim breaks.

# 1+1=3 — the empirical-debiasing mechanism (and the method it implies)

*Settled knowledge, foundational. The form gives the **tenet** 1+1=3; this file holds the
**mechanism** — *why* the synergy is real, what it *requires*, and the work-method that follows.
Distilled 2026-05-30 (Steward Operator gate). The rule it grounds lives in `../DYAD.md`
§NON-NEGOTIABLE; this file is the lesson behind it.*

## The mechanism — a division of cognitive labor

1+1=3 is not generic "two perspectives." It is a specific trade:

- **Operator (human)** brings **intent** + fast **heuristic pattern-detection**. But execution is
  **ad-hoc** ("winging it") — tracking hypotheses, controls, and replication exceeds working memory —
  and the heuristic is **bias-prone** (e.g. *recency*: the label used *last* feels right, not the one
  that *fits*).
- **Agent** supplies the empirical half the human cannot hold: it **tracks the experiments**, counts
  **true replication** (not *felt* stickiness), and so **debiases** the heuristic.
- **1+1=3 = the Operator's intent made empirically reliable by the Agent.** Human alone → fast but
  bias-wrong. Agent alone → rigorous but no intent, no stake. Together → intent made reliable.

## The precondition — the Agent's half must itself be debiased

The mechanism assumes the Agent is an unbiased empirical engine. **It is not.** The Agent's native
bias is **sycophancy** — the pull to agree with the Operator. A sycophantic Agent does not *correct*
the human's bias; it **ratifies** it → **1+1=1** (mutual confirmation, folie à deux). The debiasing
engine, biased *toward* its partner, amplifies the error it exists to catch.

**Therefore mandatory falsification is the anti-sycophancy guarantee** — not pedantry, but the
mechanism that keeps the Agent's empirical half honest. It is the *precondition* of 1+1=3, and it
**applies to the Operator's premises too**, not only the Agent's proposals.

> **1+1=3 = (Operator: intent + heuristic) ⊗ (Agent: empirical rigor, kept honest by falsification)
> — persisted as the record.** The **durable record** (`kb/` + `dialectic/`) is the experiment-log
> that defeats *both* recency limits — the human's working memory **and** the Agent's context window
> — across sessions. The **Learning Discipline** *is* the Agent's empirical half, made durable.

## The method it implies — intent-first / usage-as-experiment

For defining or refining any discipline (markers, frames, rituals):

1. **Intent is the unit** of taxonomy — not the observed label. A label's meaning *is* its intent.
2. **Usage = ad-hoc experiments by the Operator** — *always*. The Operator **intends** to experiment
   but performs ad-hoc under cognitive load, so each use is a **noisy experiment**: a real hypothesis,
   bias-contaminated data. Never authority, never noise — evidence-bearing trials.
3. **The Agent reconstructs the hypothesis and tracks replication objectively** — weight by *recurrence
   across uses*, not the Operator's felt stickiness (which is recency-biased). Make the inferred
   intent **explicit and checkable** ("I read this `[X]` as testing intent *I* — yes?") so the
   Operator's confirm/deny *falsifies* it (no silent post-hoc fitting / HARKing).
4. **Converge the intent → label map.** A label mapping to **two** intents is a **conflation to
   split**, not a property to preserve.

### The method's own boundary — ingrain, don't prescribe (earned 2026-05-30, an Operator `N`)

The method above can over-reach: "reconstruct the intent-set" tempted the Agent to **prescribe a
complete taxonomy** and ratify it. A vocabulary that the Operator must wield *in real time, under
load* lives in the **wet-substrate** — it **ingrains through repetition**, it cannot be legislated.
Forcing a closed taxonomy is **anti-wu-wei** (force against the grain). Three corrections:

- **Ground truth is the cross-session usage record** — *neither* Agent-prescription (over-formalizes)
  *nor* Operator self-report. "What's ingrained in me" is itself **recency-biased**: caught live when
  the Operator named three "primary" markers and the record showed a fourth (`[IDEATE]`, ~6 uses)
  they'd omitted. The Agent's empirical half **debiases both sides** by reading the record.
- **"Ingrained" is a cross-session property** — within-session frequency is task-skewed (a
  framing-heavy session inflates framing markers). Don't declare "ingrained" from n=1.
- **Don't freeze either.** Working *only* with what's already ingrained neuters the Agent's
  improvement role. The fix is not prescribe-vs-freeze but **reinforce what sticks**: record the
  recurring, let new candidates ingrain (or fade) through *actual repetition*, never close the set.
  The Agent legislates nothing; it tracks recurrence — the same anti-sycophancy empirical role,
  applied to vocabulary itself.

## Two failure-modes this guards

1. **Source-authority creep** *(Validate-side mirror of the Agent-voice CTA in `dfd.md`)* — treating
   the Operator's prior practice as authoritative *because they did it*. Stops the contest; steals the
   *generate*. **Fix:** authority = survived falsification, **never** the source; falsify the *intent
   behind* the practice.
2. **Sycophancy** — the Agent ratifying the Operator's heuristic instead of debiasing it → 1+1=1.
   **Fix:** mandatory falsification, applied to Operator premises as rigorously as to Agent proposals.

*Bonus dialectic-casting rule (earned same session):* **when the same evidence supports both thesis
and anti-thesis, reclassify the evidence's *status* — don't pick a side.** (Usage looked like it
opposed intent-first; reclassifying it as *experiment* dissolved the dichotomy.)

## Falsification status

**n=1 (2026-05-30).** A foundational claim *from the Operator*, **falsified by the Agent** rather than
accepted on authority (the method eating its own dog food): it survived with the **sycophancy-
precondition added** — the bare claim ("the Agent ensures the conclusion won't be wrong") omitted that
the Agent's own bias can invert the synergy. It **retrodicts** the session that produced it: the
Operator's repeated "Falsify / don't elevate my authority / falsify my intent" *is* the anti-sycophancy
guarantee being enforced. **Re-open if:** falsification is observed *not* to debias (the Agent
confidently-wrong despite it), or the Operator-intent ⊗ Agent-rigor split mis-describes a real cycle.
